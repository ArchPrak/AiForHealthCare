The request should include the user's access token in the `Authorization` header.

There are examples of how to do this in any language in the
[measurement endpoint documentation](api-reference/save-measurements).

However, here is an example using javascript:

```javascript


async function postMeasurements(fdaiUser, measurements) {
  const response = await fetch(`https://safe.fdai.earth/api/v1/measurements`, {
    method: 'POST',
    headers: {
      'Content-type': 'application/json',
      'Authorization': `Bearer ${fdaiUser.accessToken}`
    },
    body: JSON.stringify({
      measurements
    })
  });
}
```
